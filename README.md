# bruehlmann.io

Personal website of Florian Brühlmann, hosted on GitHub Pages at [bruehlmann.io](https://bruehlmann.io).

## Structure

Single-file static site — no build tools, no dependencies.

- `index.html` — the entire site (HTML, CSS, JS)
- `avatar.jpg` — profile photo
- `publications.bib` — full BibTeX reference file
- `.github/workflows/hugo.yml` — GitHub Actions deployment workflow

## Deployment

Pushing to `master` automatically deploys to GitHub Pages.
